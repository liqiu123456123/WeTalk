## 关于本项目

这里放截图
这是一个网页端的通信工具，基于WebSocket协议实现，支持文字/语音/视频三种方式通信，用户可选择使用
邮箱或者人脸识别登录。

以下是项目的一些特点：
* 人脸识别注册登录，可实现"免账号"使用，安全防盗
* 文字/语音/视频三种通信方式选择
* 基于网页端实现，手机/电脑均可使用，无需下载客户端
* 有兴趣欢迎加入一起开发
* 联系方式在页面底部
